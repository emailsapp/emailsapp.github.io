A Github hosted static site which lists everything there is to know abouts emails

Hosted on : www.emails.app

Description

A static site using any static site generator such as Sphinx or Pelican or any other flavor in any popular language which provides the user with all the information about the following:

-Top Free email service providers as well as hosted email service providers (with details like privacy details, pricing and all possible relevant details(past hacks eg: yahoo etc) etc) such as Gmail, fastmail, Protonmail, icloud mail, zoho, migadu etc

  • Similar data for apps for platforms such as iOS, Android etc
  • Similar data for self hosted email services
  • And so on..

How it will work

Every contributor who chooses to work on this project through Gitcoin will get write access on a branch and all the commits will be made to this repository and in the end, the completed branch(or the winning bounty) will be merged with the master and hosted on the static site.

Suggestions are welcome to improve the scope of the work or the idea.
